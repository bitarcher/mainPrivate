This set of classes demonstrates how to integrate a SceneManager into a project along with ManagedScenes and ManagedLayers that utilize the ResourceManager.

Installation:
1. Import all of the classes into your project: ApplyingSceneManager, GameLevel, MainMenu, ManagedGameScene, ManagedLayer, ManagedMenuScene, ManagedScene, OptionsLayer, ResourceManager, and SceneManager.
2. Replace your AndroidManifest.xml's startup activity with ".ApplyingSceneManager"
